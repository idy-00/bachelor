# Bachelor Game

Bachelor est un jeu interactif en temps reel concu pour etre joue entre amis. Le projet utilise une architecture simple et efficace basee sur PHP pour le backend et JavaScript (jQuery) pour le frontend, permettant une synchronisation fluide entre les joueurs via un systeme de polling.

## Fonctionnalites

- Lobby en temps reel : Rejoignez une salle d'attente et voyez les autres joueurs se connecter.
- - Systeme de sessions : Reprise automatique de session grace a une API de verification d'identite.
  - - Deroulement par rounds : Soumettez vos reponses et visualisez les resultats a la fin de chaque manche.
    - - Interface dynamique : Transition fluide entre le lobby, le jeu et les resultats sans rechargement de page.
     
      - ## Stack Technique
     
      - - Frontend : HTML5, CSS3, JavaScript (jQuery)
        - - Backend : PHP (API REST simplifiee)
          - - Base de donnees : Fichiers JSON (stockes dans le dossier data/)
